Production-ready multi-agent orchestration platform, FastAPI backend and Next.js operations dashboard.

## Why Hivemind

Hivemind routes research and execution workflows across specialized agents, keeps traceability, and exposes operational metrics in real time.

## Quickstart

```bash
python -m venv .venv && source .venv/bin/activate
pip install -e .[dev]
uvicorn ai_agent_demo.main:app --reload
```
